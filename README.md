# LampMarket. Домашнее задание 4

Frontend-составляющая интернет-магазина лампочек на React.

Проект создан для ДЗ4 на базе пользовательского интерфейса из ДЗ3. Следующий этап работ: подключить Redux, заменить тестовые данные на HTTP-запросы через `fetch` и связать приложение с микросервисами товаров и заказов из ДЗ2.

## Технологии

- React
- Vite
- React Router DOM
- Redux Toolkit и React Redux
- `fetch`
- CSS

## План интеграции

- `product-service`: получение категорий, списка товаров, поиска, фильтрации и карточки товара.
- `order-service`: получение корзины, добавление товара, изменение количества, удаление позиции, оформление заказа и получение заказов.
- Глобальное состояние Redux: товары, категории, корзина, заказы, состояния загрузки и ошибки.
- Обработка состояний интерфейса: загрузка, ошибка, пустой список, конфликт остатков, недоступность сервиса.

## Запуск frontend

```bash
npm install
npm run dev
```

Локальный адрес Vite по умолчанию:

```text
http://localhost:5173
```

Если порт занят, Vite покажет в терминале другой свободный адрес.

Проверка сборки:

```bash
npm run lint
npm run build
npm run preview
```

## Backend для ДЗ4

Перед проверкой нужно поднять микросервисы из соседнего репозитория `lamp-market-dz2`:

```bash
cd ../lamp-market-dz2
docker compose up -d --build
curl -f http://localhost:8001/health
curl -f http://localhost:8002/health
```

## Страницы

| Маршрут | Страница |
|---|---|
| `/` | каталог |
| `/catalog` | каталог |
| `/products/:id` | карточка товара |
| `/cart` | корзина |
| `/checkout` | оформление заказа |
| `/order-success` | подтверждение заказа |
| `*` | страница 404 |

## Не входит в ДЗ4

- Панель управления.
- Авторизация и регистрация.
- Оплата.

Эти части исключены по условию задания.
